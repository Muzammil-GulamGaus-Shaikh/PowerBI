# Power BI Practice Repository

## Overview

This repository contains my hands-on practice work in Power BI.
It includes data cleaning, transformation using Power Query, data modeling, and dashboard creation to build real-world business insights.

## What I’m Learning

* Data cleaning and handling null / unwanted values
* Text, numerical, and date transformations
* Creating conditional columns
* Merging and appending datasets
* Data modeling (relationships, measures)
* Building interactive dashboards
* KPI reporting and trend analysis

## Tools Used

* Power BI Desktop
* Power Query
* DAX
* Excel / CSV datasets

## Repository Structure

* **Datasets** – Raw and cleaned datasets used for practice
* **Power Query Files** – Data transformation steps
* **Dashboards** – Power BI report (.pbix) files
* **Notes** – Learning notes and concepts

## Purpose

The goal of this repository is to strengthen my skills in Business Intelligence and Data Analytics by applying concepts to practical scenarios.

## Status

Ongoing — I update this repository as I learn new Power BI features and build new dashboards.
